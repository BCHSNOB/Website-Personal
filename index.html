Penjelasan Properti CSS

Berikut adalah penjelasan tentang properti CSS yang digunakan dalam proyek ini:

  • position: fixed - Membuat elemen tetap di tempat saat halaman di-scroll
  • top / bottom / left / right - Mengatur jarak elemen dari sisi layar
  • z-index - Menentukan urutan tumpukan elemen
  • display: flex - Mengaktifkan layout Flexbox
  • flex-direction: column / column-reverse - Menyusun elemen secara vertikal (atas ke bawah / bawah ke atas)
  • justify-content - Mengatur posisi elemen di sumbu utama (horizontal/vertikal tergantung arah)
  • align-items - Menyelaraskan elemen di sumbu silang (biasanya horizontal)
  • gap - Menentukan jarak antar elemen
  • flex-wrap: wrap - Membungkus elemen jika tidak cukup ruang
  • flex: 1 1 300px - Kontrol pertumbuhan dan ukuran minimum elemen dalam Flexbox
  • width / height - Menentukan ukuran elemen
  • margin - Jarak luar antara elemen dan elemen lain
  • padding - Jarak dalam antara isi elemen dan tepi elemen
  • background: linear-gradient - Gradasi warna latar belakang
  • color - Warna teks
  • border-radius - Membuat sudut elemen membulat
  • box-shadow - Menambahkan efek bayangan pada elemen
  • text-shadow - Efek bayangan pada teks
  • text-align - Menentukan perataan teks
  • font-size - Ukuran huruf
  • font-family - Jenis font yang digunakan
  • font-weight - Ketebalan huruf
  • white-space: nowrap - Mencegah teks turun ke baris baru
  • transition - Membuat efek perubahan halus saat properti berubah
  • transform - Mengubah bentuk elemen (perbesar, geser, putar, dll)
  • :hover - Efek saat pointer berada di atas elemen
  • cursor: pointer - Mengubah kursor jadi tangan saat diarahkan ke elemen
  • opacity - Mengatur transparansi elemen
  • pointer-events: none - Membuat elemen tidak merespons interaksi mouse
  • animation - Menjalankan animasi pada elemen
  • @keyframes - Mendefinisikan animasi kustom
  • scroll-behavior: smooth - Membuat scroll halaman lebih halus
  • text-decoration: none - Menghilangkan garis bawah pada teks/link
  • display: grid - Mengaktifkan sistem layout grid
  • grid-template-columns - Menentukan jumlah dan ukuran kolom dalam grid
